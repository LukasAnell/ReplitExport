The motherboard is another important part of the system. Everything is connected to it, drives, CPU, GPU, RAM, case, fans, you name it. It is the main connection for communications between different parts. On the motherboard, there is a dedicated place for the CPU and RAM, while the GPU can go in most of the PCIe slots. There are usually multiple PCIe slots on a motherboard, meaning you can have multiple GPUs, though it is only feasible for some kinds of workloads.
